Recent COVID-19 Exposures Meet-in-the-Middle Feature Open Businesses in Flint

Flint COVID-19 Exposure Mapping

An Implementation of the Google Maps API and MapQuest API to map out and display relevant Covid-19 exposures and
the updated Business hours around Flint, Michigan

A CIS-363 Project by Ian Czech, Karim Elagamy, and Tadziu Kosiara

Mission Statement

We are a group of UM-Flint Students who wanted to create a relevant, useful tool for the community in the midst of this global pandemic. We have incorporated the Google Maps API as well as the MapQuest API to provide you with some useful features. These include active, real time mapping of recent COVID-19 exposures around Flint and the Genesee County area, as well as a list of open businesses with their updated hours, a meet-in-the-middle feature for convenience, and a live map that integrates your Google calendar events and displays them on the map as well as computing the necessary commute time from location to location. This is done using the Google Maps API with their live traffic updates.
